PayeeLocationService

public class PayeeLocationService : Service

Provides access to payee location operations

  • getPayeeLocations(budgetId:completion:)

    Returns a list of all payee locations.

    • budgetId: The id of the budget (last_used can also be used to specify the last used budget)
    • completion: Completion handler that takes in two parameters. First parameter contains the list of payee locations, and the second parameter contains any errors encountered during retrieval.

    • payeeId: The id of the payee
    • completion: Completion handler that takes in two parameters. First parameter contains the payee locations, and the second parameter contains any errors encountered during retrieval.
